# Prassanna Fashion Designs and Institute Pvt. Ltd

## Project Info

Professional fashion education, tailoring & design training for women.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```

## Deployment

Build the project for production:
```bash
npm run build
```
